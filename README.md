# only-dog-fe

Project for learning

## Install / Update

```
yarn
```

## Start

```
yarn start
```

## Deploy

- Change homepage in package.json, VITE_DOMAIN_API in .env and domain API in CNAME

### For Linux user

```
yarn buildAndDeploy
```

### For Windows user

```
yarn build
```

- Copy file CNAME to folder dist

```
yarn deploy
```
